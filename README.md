# Expense Tracker

Expense tracker created using React.js.

## Features
- Add new expenses
- Filter expenses by year
- Chart visualising expenses

## How to run

### Prerequisites
- Node
- npm

### How to run

1. Clone this repo
2. Run npm install
3. Run npm start
